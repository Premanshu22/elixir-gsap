## 📌 Overview

**ELIXIR** is a premium, scroll-based cocktail website designed to showcase advanced frontend animation techniques using GSAP.  

The project focuses on immersive storytelling, smooth scroll interactions, pinned sections, text reveals, and dynamic masking effects.

## ⚙️ Tech Stack

- **React** – Component-based UI structure
- **GSAP** – ScrollTrigger, timelines, masking, and motion design
- **Tailwind CSS** – Utility-first styling
- **Vite** – Lightning-fast development and optimized production builds

---

## 🔥 Key Features

- Scroll-triggered animations
- Split text reveals
- Pinned sections
- Image masking transitions
- Custom animated carousel
- Smooth parallax effects
- Fully responsive design
- Optimized animation performance
